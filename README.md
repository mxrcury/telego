### Build

In order to use this application you need to have installed `Go`

Environmental variables that are required:

```env
API_ID="example"
API_HASH="example"
```

##### Windows

Use the next command to build it:

```sh
$ go build -o "name-of-build"
```

And then run executable file.

##### Linux and MacOS

Use the next command to build it:

```sh
$ make
```
